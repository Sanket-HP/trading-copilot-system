# Trading Copilot System

AI-powered trading backend system built using FastAPI.

## Features

- JWT Authentication
- Live Stock Market APIs
- Historical Market Data
- RSI Indicator
- MACD Indicator
- AI Trading Signals
- Watchlist System
- Portfolio Management
- Profit/Loss Tracking

## Tech Stack

- FastAPI
- SQLite
- SQLAlchemy
- JWT Authentication
- yFinance
- Pandas
- TA Library
